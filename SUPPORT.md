# Support

CPEB Equipment Booking System is an experimental educational project.

## Where to Ask

Use GitHub issues for:

- reproducible bugs;
- clear feature requests;
- documentation problems;
- project setup problems.

## Before Opening an Issue

Please check:

- `README.md`;
- `docs/reproducibility.md`;
- existing issues;
- current limitations in `docs/limitations.md`.

## What Not to Share

Do not include:

- passwords;
- API tokens;
- `.env` values;
- personal documents;
- real student data;
- private screenshots;
- confidential university files.

## Response Expectations

There is no guaranteed response time. This is not a commercial support channel.
