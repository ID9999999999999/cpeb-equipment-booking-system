# Code of Conduct

## Our Standard

Participants are expected to communicate respectfully and constructively.

Examples of positive behavior:

- using welcoming and inclusive language;
- accepting constructive feedback;
- focusing on what is best for the project;
- showing empathy toward other contributors.

Unacceptable behavior includes harassment, insults, threats, doxxing, or sharing private information.

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, documentation, and review comments.

## Enforcement

For this educational project, the repository owner is responsible for moderation and may remove comments, close issues, block users, or reject contributions that violate this policy.
