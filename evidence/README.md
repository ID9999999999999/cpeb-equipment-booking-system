# Evidence

Use this directory to store proof that the project works.

Good evidence includes:

- screenshots without private data;
- demo GIFs or short videos;
- test output logs;
- API request/response examples;
- architecture diagrams;
- benchmark notes if benchmarks are real and reproducible;
- before/after notes.

Do not add:

- passwords;
- API tokens;
- real `.env` files;
- real student data;
- private screenshots;
- confidential university documents.
